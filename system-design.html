Key Data Flows

Training Flow

1Generate 10k synthetic churn records
2Validate schema & save to data/raw/YYYY-MM-DD/
3StandardScaler + OneHotEncoder → fit on train
4Train RandomForest with MLflow autolog
5Evaluate AUC, F1, confusion matrix → log artifacts
6Register to MLflow registry → Staging if AUC > 0.75

Serving Flow

1Load Production model from MLflow on startup
2POST /predict → Pydantic validation
3Pipeline predicts (preprocessor + classifier)
4Return { prediction, probability, model_version }
5Log prediction to data/predictions.csv (UUID)
6POST /feedback → write actual label back to CSV

Drift & Retrain Flow

1Cron (1am daily) loads last 24h predictions
2Compare vs reference dataset (training data)
3Generate Evidently HTML reports
4drift_share > 0.3 → Slack alert + trigger DAG
5Retrain on historical + labeled production data
6Champion/challenger (+1% AUC) → promote or reject
